Simulation de la loi binomiale avec Python¶

Auteur : Franck CHAMBON

Un document issus de cet ensemble de cours pour Python et les mathmatiques.

On rappelle que la loi binomiale modlise le nombre de succs obtenus lors de la rptition indpendante de plusieurs expriences alatoires identiques.

Ces expriences alatoires identiques et indpendantes peuvent tre considres comme des preuves de Bernoulli.

Commenons par simuler ces preuves ; on fera ensuite le compte des succs en cas de rptition.

Simulation d'épreuve de Bernoulli¶

Python propose le module random qui fournit, en particulier les fonctions randint et random, gnrateurs de nombres pseudo-alatoires.

  • randint(a, b) : retourne un entier entre a inclus et b inclus.
  • random() : retourne un flottant entre $0$ inclus et $1$ exclu. Simule un rel.

On peut s'en servir pour simuler une exprience de Bernoulli o la probabilit de succs est connue. Deux cas sont envisager, la probabilit de succs est :

  • une fraction $\frac a b$ ;
  • ou bien une valeur relle $p$.

La ligne ci-dessous est inclure une fois pour toute au dbut d'un code Python, les fonctions sont alors ensuite accessibles.

Il s'agit d'une situation d'quiprobabilit (le d est non truqu), avec $3$ issues favorables parmi $20$ issues possibles.
La probabilit de succs est $\frac 3 {20}$.

Exemple 2 : probabilité de succès, un nombre réel¶

Dans un jeu de flchette, Tom envoie au hasard une flchette dans une zone qui contient une cible.

Tom russit toujours envoyer sa flchette dans la zone carre de $2~\textrm{m}$ qui a donc une aire de $4~\textrm{m}^2$, mais de faon alatoire uniforme dans la zone.
Dans cette zone, il y a une cible, un disque de $1~\textrm{m}$ de rayon, et dont l'aire est donc $\pi~\textrm{m}^2$. La probabilit de succs pour toucher la cible est donc $\frac \pi 4$.

Simulation d'un schéma de Bernoulli¶

On rpte maintenant plusieurs fois des expriences de Bernoulli identiques et de manire indpendante, et on compte le nombre de succs. On obtient un schma de Bernoulli.

Compter une fois le nombre de succès¶

On considre une exprience de Bernoulli de paramtre $p = 0.2$, et on rpte $n = 100$ fois cette exprience de manire indpendante.

Comptons le nombre de succs :

Il est raisonnable de trouver ici un nombre de succs proche de $p\times n = 0.2\times 100 = 20$.

On va rpter plusieurs fois ce schma et afficher le nombre de succs.

Compter plusieurs fois le nombre de succès¶

In [9]:
def simulation_schma(p, n):
    "Retourne le nombre de succs dans un schma de Bernoulli de paramtres p et n"
    nb_succs = 0 # initialisation
    for _ in range(n):
        if random() < p:
            nb_succs += 1 # augmentation de 1
    return nb_succs

Comparer les statistiques et les probabilités¶

On pourrait comparer les frquences obtenues avec les probabilits attendues.

D'abord rappelons (cours de maths) que dans un schma de Bernoulli de paramtres $p$ et $n$,

En notant $X$ la variable alatoire du nombre de succs,
la probabilit d'obtenir $k$ succs est $\mathbb P(X = k) = \binom n k p^k (1-p)^{n-k}$

partir de la version Python 3.8, on pourra remplacer les 3 premires lignes par from math import comb

comb(n, k) calcule le coefficient binomial $\binom n k$.

In [15]:
from math import factorial
def comb(n, k):
    return factorial(n) // factorial(k) // factorial(n-k)

plt.plot([comb(n, k) * p**k * (1-p)**(n-k) for k in range(0, 40)], "ro")
plt.show()

%matplotlib inline
import matplotlib.pyplot as plt
from random import random
from math import sqrt, factorial # , comb #  partir de Python 3.8
def comb(n, k): return factorial(n) // factorial(k) // factorial(n-k)

def simulation_schma(p, n):
    "Retourne le nombre de succs dans un schma de Bernoulli de paramtres p et n"
    # rcriture avec un style fonctionnel
    return sum(1 for _ in range(n) if random() < p)


#####  valeurs  modifier pour d'autres tests
p = 0.1
n = 500
nb_rep_schem = 10000
hw = 3 # demi-largeur en sigma pour l'intervalle

##### fin des modifications  faire



rsultats = [simulation_schma(p, n) for test in range(nb_rep_schem)]

mean = n * p  # esprance
variance = n * p * (1-p)
sigma = sqrt(variance)

minI = int(mean - hw * sigma)
maxI = int(mean + hw * sigma)

plt.plot(range(minI, maxI), [comb(n, k) * p**k * (1-p)**(n-k) for k in range(minI, maxI)], "ro")
plt.hist(rsultats, range = (minI, maxI), bins = maxI - minI, density=True)
plt.xlabel("Nombre de succs")
plt.ylabel("Frquence")
plt.title("Comparaison statistique - probabilit")
